# Offre Formation IA - Challenge 100 jours LN-IA

Ce dépôt contient la page publique partageable de l'offre Challenge 100 jours de l'Atelier LN-IA.

Version enrichie : la page présente l'affiche, la promesse, les livrables, le programme détaillé en 5 phases, la soutenance finale avec attribution de prix symboliques, les priorités d'accès, la FAQ courte et l'appel à rejoindre le groupe WhatsApp.

## Fichiers

- `index.html` : page publique prête pour GitHub Pages, placée à la racine du dépôt.
- `assets/affiche-challenge-100jours-ln-ia-qr.png` : affiche complétée avec QR code utilisée dans la page.
- `assets/affiche-challenge-100jours-ln-ia-completee.png` : affiche complétée conservée comme référence.
- `assets/affiche-challenge-100jours-ln-ia.jpeg` : première affiche conservée comme référence.

## Publication GitHub Pages

Pour publier cette page depuis GitHub Pages :

1. ouvrir les paramètres du dépôt GitHub ;
2. aller dans `Pages` ;
3. choisir la branche `main` ;
4. choisir le dossier `/root`.

URL attendue :

